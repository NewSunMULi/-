# 我们要研究什么

我们的主题是关于游戏抽卡的概率问题，为什么我们要研究这个呢？因为当今带抽卡系统的游戏越来越多，游戏抽卡问题登上各大新闻的次数也越来越多，为了让人们合理、理性规划抽卡，揭示不同种类的游戏抽卡模型和概率模型，所以我们尝试研究游戏抽卡。

# 研究内容

我们取5个比较具有代表性、有着不同抽卡类型的游戏进行研究，并从中总结大部分游戏抽卡的规律。
## 研究对象
1.原神 ---- 双保底，概率动态变化
2.崩坏三 ---- 单保底，概率静态变化
3.明日方舟 ---- 单保底，概率动态变化
4.明日之后 ---- 无保底，极低概率
5.Apex英雄 ---- 单保底，保底所需次数大，一次多抽
